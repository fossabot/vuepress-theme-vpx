一款基于 VuePress 开发的主题

  • 快速上手 →

# 安装主题

    yarn add @qcyblm/vuepress-theme-vpx -D

    npm i @qcyblm/vuepress-theme-vpx -D

    cnpm i @qcyblm/vuepress-theme-vpx -D

    // Make sure to add code blocks to your code group

    # 引用主题

    // .vuepress/config.js

module.exports = {
  ...
  theme: '@qcyblm/vpx',
  ...
}
