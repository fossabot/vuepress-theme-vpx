@qcyblm/vuepress-theme-vpx

一款基于 VuePress 开发的主题

快速上手 →

# 安装主题

yarn add @qcyblm/vuepress-theme-vpx -D
# or npm i @qcyblm/vuepress-theme-vpx -D

# 引用主题

// .vuepress/config.js

module.exports = {
  ...
  theme: '@qcyblm/vpx',
  ...
}

注意

请确保你的 Node.js 版本 >= 8.6。
